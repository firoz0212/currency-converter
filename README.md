# Currency Converter

A responsive Currency Converter application built using HTML, CSS, and JavaScript.
The application allows users to convert one currency into another using real-time exchange rates.

## Features

* Convert currencies instantly
* Real-time exchange rate data
* Clean and responsive user interface
* Easy to use

## Technologies Used

* HTML
* CSS
* JavaScript
* Exchange Rate API

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Start converting currencies
